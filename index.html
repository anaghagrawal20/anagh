The Basic Language of the Web: HTML

posted by laura jones on monday , june 21st 2027

all modern language of the web application are built using three fundamental technologies: HTML, CSS. and JavaScript. these are the languages of the web.

in this post, lets focus on HTML. we will learn what HTML is all about, and why you too should learn it.

What is HTML?

HTML stands for Hyper Text Markup Language, its a markup language that web developres use to structure and describe the content of a web page (not a programing language).

HTML consits of elements that describes diffrent types of content: paragraps, links, heading, images, video, etc. web browers understand HTML code as websites.

In HTML, each element is made up of 3 parts:

  1. the opening tag
  2. the closing tag
  3. the actual element

you can learn more at MDN

why should you learn HTML?

there are countless reasons for learning the fundamental language of the web. here are 5 of them:

  • to be able to use the fundamental web dev language
  • to hand-craft beautiful website insted of reyling on tools like worpress or wix
  • to build web application
  • to impress friends
  • to have fun
